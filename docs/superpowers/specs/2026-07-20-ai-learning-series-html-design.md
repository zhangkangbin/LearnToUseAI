# AI 学习系列 HTML 阅读站设计

## 目标

将《普通人如何高效使用 AI》的基础篇、进阶篇和精通篇转换为无需联网即可打开的 HTML 阅读站，提升长文浏览、定位和跨设备阅读体验。

## 交付物

- `index.html`：三篇文章的入口页。
- `普通人如何高效使用AI-基础篇.html`
- `普通人如何高效使用AI-进阶篇.html`
- `普通人如何高效使用AI-精通篇.html`

## 页面结构

入口页展示系列定位、阅读顺序和三张文章卡片。文章页由顶栏、目录、正文和相邻文章导航组成：

- 顶栏包含返回入口、主题切换和阅读进度。
- 桌面端目录固定在左侧；移动端以可展开抽屉呈现。
- 目录根据当前阅读位置高亮对应标题。
- 页末按“基础篇 → 进阶篇 → 精通篇”提供上一篇和下一篇。

## 内容转换

以三个现有 Markdown 文件为唯一内容来源，保留标题、段落、列表、引用、表格、链接、强调和围栏代码块。为各级标题生成稳定锚点，供目录和深链接使用。原始 Markdown 不会被修改。

## 视觉与交互

- 支持浅色、深色及跟随系统三种主题；用户选择写入浏览器本地存储。
- 显示页面顶部阅读进度条。
- 提供适合中文长文阅读的字重、行距、最大行宽和代码/表格横向滚动。
- 所有样式和脚本内嵌，无第三方依赖、无网络请求。

## 异常与验证

转换时缺少任何原始 Markdown 文件即中止并报错。生成后检查所有四个 HTML 均存在、UTF-8 编码正确，且文章标题、目录标题、主题控件和相邻文章链接均已写入。使用本地浏览器或静态解析检查页面的基础结构与链接。
